# سجل تتبع الأخطاء والتشخيص (Bugs & Diagnostics Log)

أي خطأ أو فشل برمجي يتم رصده بواسطة أداة الـ Sandbox يتم تدوينه هنا فوراً مع تحليل أسبابه وجذر المشكلة التقنية لمنع تكرارها في المستقبل.

---

## مفتاح الأولوية (Priority Legend)

| الرمز | المستوى | الوصف |
| :---: | :--- | :--- |
| 🔴 | حرج (Critical) | خلل يمنع تشغيل النظام أو يُشكّل ثغرة أمنية فورية — يجب إصلاحه قبل أي مهمة أخرى |
| 🟡 | متوسط (Medium) | خلل يؤثر على تجربة المستخدم أو يُعطّل ميزة فرعية — يُجدوَل للإصلاح في أقرب دورة |
| 🟢 | منخفض (Low) | خلل تجميلي أو تحسيني لا يُعيق العمل — يُعالَج عند توفّر الوقت |
